# Feywild Time Warp
*Source: Dungeon Master's Guide (2024) p. 196* 

**Feywild Time Warp**

| dice: 1d20 | Result |
|------------|--------|
| 1–2 | Days become minutes |
| 3–6 | Days become hours |
| 7–13 | No change |
| 14–17 | Days become weeks |
| 18–19 | Days become months |
| 20 | Days become years |
