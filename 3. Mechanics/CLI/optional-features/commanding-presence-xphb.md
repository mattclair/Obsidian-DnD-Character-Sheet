# Commanding Presence
*Source: Player's Handbook (2024) p. 95*  

When you make a Charisma ([Intimidation](3.%20Mechanics/CLI/rules/skills.md#Intimidation), [Performance](3.%20Mechanics/CLI/rules/skills.md#Performance), or [Persuasion](3.%20Mechanics/CLI/rules/skills.md#Persuasion)) check, you can expend one Superiority Die and add that die to the roll.
