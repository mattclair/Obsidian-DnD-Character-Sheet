# Ambush
*Source: Player's Handbook (2024) p. 94*  

When you make a Dexterity ([Stealth](3.%20Mechanics/CLI/rules/skills.md#Stealth)) check or an [Initiative](3.%20Mechanics/CLI/rules/variant-rules/initiative-xphb.md) roll, you can expend one Superiority Die and add the die to the roll, unless you have the [Incapacitated](3.%20Mechanics/CLI/rules/conditions.md#Incapacitated) condition.
