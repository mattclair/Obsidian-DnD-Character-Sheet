# Hostile [Attitude]
*Source: Player's Handbook (2024) p. 368. Available in the <span title='Systems Reference Document (5.2)'>SRD</span> and the Free Rules (2024)* 

A Hostile creature views you unfavorably. You have [Disadvantage](3.%20Mechanics/CLI/rules/variant-rules/disadvantage-xphb.md) on an ability check to influence a Hostile creature.
