# Death Saving Throw
*Source: Player's Handbook (2024) p. 365. Available in the <span title='Systems Reference Document (5.2)'>SRD</span> and the Free Rules (2024)* 

A [player character](3.%20Mechanics/CLI/rules/variant-rules/player-character-xphb.md) must make a Death Saving Throw (also called a Death Save) if they start their turn with 0 [Hit Points](3.%20Mechanics/CLI/rules/variant-rules/hit-points-xphb.md).
