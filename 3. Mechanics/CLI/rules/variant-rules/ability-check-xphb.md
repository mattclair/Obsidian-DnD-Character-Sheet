# Ability Check
*Source: Player's Handbook (2024) p. 360. Available in the <span title='Systems Reference Document (5.2)'>SRD</span> and the Free Rules (2024)* 

An ability check is a [D20 Test](3.%20Mechanics/CLI/rules/variant-rules/d20-test-xphb.md) that represents using one of the six abilities—or a specific skill associated with an ability—to overcome a challenge.
